## 💚 Sobre o projeto

Este projeto foi desenvolvido como um **checkpoint da faculdade** na matéria de **PIRP (Pensamento e Resolução de Problemas)**.

A proposta era criar um sistema em Python capaz de **registrar e classificar ações sustentáveis**, utilizando lógica de programação e interação com o usuário via terminal.

---

## 🎯 Objetivo da atividade

Aplicar conceitos como:

- Entrada e saída de dados  
- Estruturas condicionais (`if`, `elif`, `else`)  
- Validação de dados  
- Lógica de decisão  

Tudo isso dentro de um contexto simples e prático.

---

## 🖥️ Como funciona

O programa pergunta ao usuário:

- Qual ação sustentável foi realizada  
- O nível de impacto (1 a 3)  
- Quantas vezes foi feita no mês  
- Se foi validada  
- Se foi uma ação coletiva  

Com base nessas respostas, ele classifica a ação como:

- Participação  
- Pontuação bônus  
- Destaque sustentável  
- Ou sem classificação (caso não validada)
